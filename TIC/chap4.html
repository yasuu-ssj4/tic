résumé chapitre 4 - Internet et Web

1) Introduction :

Le web et l'internet c'est pareil:

Le WWW est un système hypertexte public sur Internet qui permet de consulter des pages sur des sites grâce à des hyperliens. Le Web est une application d'Internet et a été inventé par Tim Berners-Lee et Robert Cailliau. Internet est un support de communication général et le Web représente une partie des contenus circulant sur Internet.

Principe World Wid Web:

Il permet d'accéder à des documents textuels hébergés sur des machines connectées à Internet et reliés entre eux par des liens hypertexte. Aujourd'hui, le Web sert à fournir des ressources de différentes natures, y compris du contenu interactif pour des activités telles que l'achat ou la discussion en ligne.

URL (pour Uniform Resource Locator):

Un lien hypertexte est un élément cliquable qui dirige l'utilisateur vers une nouvelle page ou ressource en suivant une URL. Cette dernière fournit l'adresse du serveur et le chemin d'accès à la page ou à la ressource.

2) Aspects techniques du Web:

Le Web est basé sur trois mécanismes fondamentaux :

les URL pour localiser les ressources, le protocole HTTP pour échanger des informations entre les clients et les serveurs, et le HTML pour décrire les pages Web, y inclure des objets et créer des liens hypertexte entre les pages.

3) Page web et site web:

Une page web

Un document multimédia contenant des images, du texte, des interactions et des liens. Elle peut etre décrite avec des langages comme HTML, XHTML ou HTML5 et aussi stockée dans un fichier HTML. Elle est aussi interprétée par un navigateur et localisée sur internet via une adresse URL. Les pages web permettent d'accéder à d'autres pages en suivant des liens. Un site web regroupe des pages autour d'une thématique commune, liées par des liens hypertexte. Les sites web sont généralement hébergés sur un même serveur et structurés sous forme d'arborescence de dossiers et de fichiers, permettant l'utilisation de chemins relatifs.

4) Création d’une page web HTML:

HTML (HyperText Markup Language)

Un langage de description de pages web. Un document HTML est un fichier texte structuré à l'aide de balises qui délimitent différentes parties du document et sont interprétées par les navigateurs web comme Mozilla Firefox, Internet Explorer et Safari. Pour créer un document HTML, on peut utiliser un éditeur de pages web comme Dreamweaver ou un éditeur de texte comme Bloc-notes ou Wordpad. Un fichier HTML doit comporter au moins quatre balises essentielles : <*html> ... <*/html> : Délimite le début et la fin du document. <*head> ... <*/head> : Contient l'en-tête du document. <*title> ... <*/title> : Spécifie le titre du document. <*body> ... <*/body> : Contient le corps du document. HTML permet également de structurer le contenu en utilisant des balises de titre (<*h1> à <*h6>) pour les titres de différents niveaux, et des balises de paragraphe ( <*p> ) pour les paragraphes.
